# **App Name**: Adoo

## Core Features:

- Organization & Branch Management: Allows organization/workspace setup, creation/editing/deactivation of branches, and assigning users to branches, with branch-scoped data visibility and global admin views.
- Role-Based Access Control: Implements RBAC with default roles (Admin, Manager, Loan Officer, Auditor) and custom roles with configurable permissions, managing access to data and settings.
- Borrower Management: Manages borrower profiles with detailed personal and financial information, document uploads, and a borrower dashboard summarizing loan activities.
- Loan Product Configuration: Configures loan products with various fields such as product name, category, loan amounts, interest rate, repayment cycle, and fees.
- Loan Creation & Financial Calculations: Facilitates loan creation with borrower and loan product selection, amount and duration input, collateral upload, and automatic calculation of total payable amount and installment schedule.
- Repayment Engine: Processes repayments, allowing single or multiple installment payments, advance payments, and partial payments, updating installment statuses and recalculating outstanding balances.
- Due-Date Monitoring Tool: Generative AI that can analyze the current risk factors from repayment histories. LLM based tool generates recommendations to take action on repayments depending on current, overdue, and upcoming schedules, as well as news of world or local events impacting repayments.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2), evoking trust, security, and financial acumen.
- Background color: A light, desaturated blue (#E5F6FD), maintaining a professional and calming visual space.
- Accent color: A contrasting yellow (#FFC857) for key actions and highlights, drawing attention to important elements.
- Headline font: 'Poppins', a geometric sans-serif providing a modern, precise feel for headings.
- Body font: 'PT Sans', a humanist sans-serif that is legible and provides warmth to the interface
- Crisp, professional icons that clearly represent financial actions and data categories.
- Clean, intuitive layout with a focus on usability and easy navigation, optimized for operational staff.
